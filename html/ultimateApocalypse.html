Ultimate Apocalypse Mod

Clean Install Steps

  1. Delete local game files on Steam for Soulstorm
  2. Check the steam apps folder and exterminatus any remains of Soulstorm
  3. Reinstall Soulstorm
  4. Boot Soulstorm and check all is a-okay
  5. Download UA 1.88.5 from here
  6. Download Tyranid Mod v0.5b2 from here
  7. Start installing UA 1.88.5
  8. Check that the install directory points to the correct directory
  9. UNCHECK the option for Tyranid Mod v0.5b2 when installing 1.88.5
  10. Download the 1.88.6.1 patch from here
  11. Copy and paste the "UltimateApocalypse_THB" and UltimateApocalypse_THB.module files into your Soulstorm directory
  12. Once finished start the install for Tyranid Mod v0.5b2
  13. Check that the install directory points to the correct directory
  14. Verify integrity of game files using the Steam wizard
  15. On your desktop you will find 4gb_patch.exe shortcut, Run that
  16. It will ask you to select an exe file in the soulstorms directory, select Soulstorm.exe
  17. Click on "Another File" button and select GraphicsConfig.exe
  18. Click OK
  19. Click "Yes" for replacing files in the same directory
  20. Boot the modmanager v1.3 from the Soulstorm directory
  21. Select UltimateApocalypse_THB ** NOT UltimateApocalypse_THB_DIDHT **
  22. Click "START GAME WITH SELECTED MOD" in the top right of the mod manager
  23. If this does not work you did it wrong, Start from step 1 do not pass go or contact friends or loved ones
