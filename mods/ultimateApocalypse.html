Ultimate Apocalypse Mod

Clean Install Steps

    Inital Clean-up

  1. Delete local game files on Steam for Soulstorm
  2. Check the steam apps folder and exterminatus any remains of Soulstorm
  3. Reinstall Soulstorm
  4. Boot Soulstorm and check all is a-okay

    5. Download Mods and Patches

  5. Download UA 1.88.5 from here
  6. Download Tyranid Mod v0.5b2 from here
  7. Download the 1.88.6.1 patch from here

    8. 1.88.5 Install

  8. Start installing UA 1.88.5
  9. Check that the install directory points to the correct directory
  10. UNCHECK the option for Tyranid Mod v0.5b2 when installing 1.88.5

    11. 1.88.6.1 Patch Install

  11. Extract the files from "UA Patch - 18861"
  12. Copy and paste the "UltimateApocalypse_THB" folder and UltimateApocalypse_THB.module file into your Soulstorm directory

    13. Tyranid Mod Install

  13. Once finished start the install for Tyranid Mod v0.5b2
  14. Check that the install directory points to the correct directory

    15. Verify Game Cache

  15. Verify integrity of game files using the Steam wizard

    16. 4gb Patch

  16. On your desktop you will find 4gb_patch.exe shortcut, Run that
  17. It will ask you to select an exe file in the soulstorms directory, select Soulstorm.exe
  18. Click on "Another File" button and select GraphicsConfig.exe
  19. Click OK
  20. Click "Yes" for replacing files in the same directory

    21. Booting the Game

  21. Boot the modmanager v1.3 from the Soulstorm directory
  22. Select UltimateApocalypse_THB ** NOT UltimateApocalypse_THB_DIDHT **
  23. Click "START GAME WITH SELECTED MOD" in the top right of the mod manager

    24. You dun goofed?

  24. If this does not work you did it wrong, Start from step 1 do not pass go or contact friends or loved ones
