The End of Days

Installation Steps

  1. Before starting, ensure that you have a clean install of C&C Generals Zero Hour
  2. Download The End of Days 094 from here
  3. Download The End of Days 094 Balance Path V3 from here
  4. Run the downloaded installer (TEOD-94.exe) as Administrator. When prompted for install location, ensure to target "/Command and Conquer Generals Zero Hour/Command and Conquer Generals Zero Hour"
  5. Delete "MultiplayerScripts.scb" and "SkirmishScripts.scb" in "/Command and Conquer Generals Zero Hour/Command and Conquer Generals Zero Hour/Data/Scripts"
  6. Unzip "TEOD_94_Balance_Patch_V3-1"
  7. Copy the contents of "TEOD_94_Balance_Patch_V3-1" into "/Command and Conquer Generals Zero Hour/Command and Conquer Generals Zero Hour", clicking "Yes" to replace existing files
  8. Download the GenTool installer exe from here
  9. Run the GenTool installer. It should detect your Generals ZH install location automatically
