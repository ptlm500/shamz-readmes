C&C: Shockwave

Installation Steps

  1. Before starting, ensure that you have a clean install of C&C Generals Zero Hour
  2. Download C&C: Shockwave 1.201 from here
  3. Unzip "ShockWaveV1201.zip"
  4. Run "ShockWaveV1201.exe", following the steps and ensuring to target it at "/Command and Conquer Generals Zero Hour/Command and Conquer Generals Zero Hour"
  5. When prompted to install the multiplayer components, follow the installer, and again target it at "/Command and Conquer Generals Zero Hour/Command and Conquer Generals Zero Hour"
  6. Download the GenTool installer exe from here
  7. Run the GenTool installer. It should detect your Generals ZH install location automatically
